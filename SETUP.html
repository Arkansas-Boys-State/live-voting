Step 1 · Set Up JSONBin (Data Store)

JSONBin is a free service that stores JSON data at a URL. Zapier writes to it; your TVs read from it.

1
Create a free account at jsonbin.io
Go to https://jsonbin.io → Sign Up (free tier is enough).
2
Create a new Bin
Click + New Bin. Paste this as the initial JSON content:
{ "nationalist": [], "federalist": [] }
3
Copy your Bin ID and API Key
The Bin ID is in the URL: jsonbin.io/b/YOUR_BIN_ID. Your API Key is under Account → API Keys.
4
Edit config.js
Open config.js and fill in binId and apiKey.

Step 2 · Set Up Google Sheets

Create one Google Sheet with two tabs — one per party.

1
Create a new Google Sheet
Name the file Mock Election Check-In.
2
Add two tabs with these exact names
Nationalist and Federalist
3
Add these column headers in Row 1 of each tab
Column A: Name  ·  Column B: Party  ·  Column C: Office  ·  Column D: Timestamp
Tip: Staff enter data starting at Row 2. Each new row triggers Zapier automatically.

Step 3 · Set Up Zapier (Two Zaps)

Create one Zap per party — they watch different sheet tabs and write to different keys in JSONBin.

Zap 1 — Nationalist Party

SettingValue
Trigger AppGoogle Sheets
Trigger EventNew Spreadsheet Row
SpreadsheetMock Election Check-In
Worksheet TabNationalist
Action AppWebhooks by Zapier
Action EventPUT
URLhttps://api.jsonbin.io/v3/b/YOUR_BIN_ID
Payload TypeJSON
Data (Body)See JSON template below
HeadersX-Master-Key → your API key  ·  Content-Type → application/json

Body template for Zap 1 — paste this into the Webhooks Data field, mapping Zapier fields:

{
  "nationalist": [
    {
      "name":  "[[Name from Google Sheets]]",
      "party": "Nationalist",
      "office":"[[Office from Google Sheets]]",
      "ts":    "[[Timestamp from Google Sheets]]"
    }
  ],
  "federalist": []
}
Note: JSONBin PUT replaces the entire bin. For a real multi-entry log you'd use a webhook to a more advanced store, but for a live confirmation display (showing just the latest entry) this approach works perfectly.

Zap 2 — Federalist Party

Identical to Zap 1, but watch the Federalist tab and swap the keys in the body:

{
  "nationalist": [],
  "federalist": [
    {
      "name":  "[[Name from Google Sheets]]",
      "party": "Federalist",
      "office":"[[Office from Google Sheets]]",
      "ts":    "[[Timestamp from Google Sheets]]"
    }
  ]
}

Step 4 · Deploy to GitHub Pages

1
Create a free GitHub account & new repository
Go to github.com → New Repository → name it mock-election → set to Public.
2
Upload all 5 files
Drag and drop into the repo: config.js, tv-nationalist.html, tv-center.html, tv-federalist.html, SETUP.html
3
Enable GitHub Pages
Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: / (root) → Save.
4
Your URLs will be:
https://YOUR-USERNAME.github.io/mock-election/tv-nationalist.html

https://YOUR-USERNAME.github.io/mock-election/tv-center.html

https://YOUR-USERNAME.github.io/mock-election/tv-federalist.html

Step 5 · Configure AbleSigns

Point each TV to its URL in AbleSigns using the "Web Page" or "URL" content type.

TVAbleSigns URL
TV 1 (Left)…/tv-nationalist.html
TV 2 (Center)…/tv-center.html
TV 3 (Right)…/tv-federalist.html
Set AbleSigns to refresh the page every 60 seconds as a fallback — the JS polls every 4 seconds automatically, but this keeps things clean if the browser ever stalls.

Step 6 · Adding Your Logo

1
Upload your logo image to the GitHub repo
Drag your logo file (PNG or SVG recommended) into the repo. Name it something simple like logo.png.
2
Edit tv-center.html
Find the comment <!-- TO ADD YOUR LOGO -->, uncomment the <img> line, set src="logo.png", and delete the placeholder div.

Day-of Workflow

1
Open the Google Sheet on a staff laptop/tablet
Both Nationalist and Federalist tabs are ready for data entry.
2
When a voter checks in, add a row to the correct tab
Fill in Name, Party, Office, and Timestamp (or use =NOW()).
3
Zapier fires within ~15 seconds
It updates JSONBin automatically — no manual action needed.
4
The correct TV lights up with the voter's name
Displayed for 30 seconds, then returns to idle. The center TV always shows the logo.
